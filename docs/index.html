Quick Start Guide

1. Install PyTunnel

pip install pytunnel

2. Start Your Local Server

# Start any local server (example with Python) python -m http.server 3000 # Or with Node.js npm start # Or with any other framework

3. Create a Tunnel

# Create tunnel to your local server pytunnel tunnel --port 3000 --server-host pytunnel.yourdomain.com # You'll get a public URL like: # https://abc123.yourdomain.com

4. Share Your URL

# Send the generated URL to anyone # They can access your local app from anywhere!
